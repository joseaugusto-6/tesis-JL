==========================================
   SISTEMA DE SEGURIDAD - REGISTRO Y RECONOCIMIENTO FACIAL
==========================================

¡Bienvenido/a!

Esta carpeta contiene la aplicación de registro y reconocimiento facial desarrollada por UCAB.

------------------------------------------
 ¿CÓMO USAR EL PROGRAMA?
------------------------------------------

1. Abre la carpeta "ReconocimientoFacialUCAB".

2. Haz doble clic en el archivo llamado:  captura-exe.exe

3. Sigue las instrucciones que aparecen en pantalla:
   - Puedes registrar un nuevo rostro.
   - Puedes probar el reconocimiento facial en tiempo real.

4. Cuando registres un rostro, se generará un archivo con tu nombre y extensión `.npy`
   (ejemplo: juan_perez.npy).
   >>> Este archivo se guardará en esta misma carpeta.

------------------------------------------
 REQUISITOS
------------------------------------------
- Windows 10 u 11.
- Cámara web conectada y funcionando.
- Permite la ejecución si tu antivirus te pregunta.

------------------------------------------
 NOTAS IMPORTANTES
------------------------------------------
- Todos los archivos necesarios para el funcionamiento están incluidos en esta carpeta.
- Los archivos `.npy` para registro del rostro en la plataforma web lo encontrarás aquí!
- Si tienes dudas o algún inconveniente, contacta con el soporte del sistema.
